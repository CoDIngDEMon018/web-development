About this project

This repository collects small web development projects, exercises, and demos. It's intended as a learning resource and portfolio for practice and sharing. Each project contains source files, brief documentation, and a live demo where possible.

Features

Responsive Layouts

Projects built with responsive design in mind — mobile-first, accessible, and adaptable.

Small & Focused

Each project covers a specific concept or UI pattern so it's easy to study and reuse.

Clear Structure

Consistent file organization and brief READMEs make the repo easy to navigate.

Get started

Clone the repo, pick a project, and run the demos locally. Contributions are welcome — open a PR with improvements or new projects.
